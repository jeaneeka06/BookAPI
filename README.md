# 📚 Book Browser Web Application

A clean, interactive front-end web application that allows users to browse books, search by title, filter by category, and view detailed information — all dynamically rendered using modern JavaScript.

This project was built as part of the Front-End Development course and demonstrates DOM manipulation, event handling, dynamic rendering, and organized project structure.

---

## 🚀 Features

### 🔍 Search
- Real-time search filters books by title as the user types.

### 🏷️ Category Filter
- Dropdown filter updates the displayed books instantly based on selected genre.

### 📘 Dynamic Book Cards
- All book cards are generated using JavaScript.
- Each card displays a cover image, title, and author.

### 📖 Book Details Panel
- Clicking a book opens a detailed view with:
  - Large cover image  
  - Title  
  - Author  
  - Category  
  - Year  
  - Description  
- Includes a close button to hide the panel.

### 🎨 Clean, Modern UI
- Styled with CSS for a polished, portfolio-ready appearance.
- Responsive layout with readable typography and consistent spacing.

1. Clone or download the repository.

3. Interact with the search bar, category filter, and book cards.

No backend or server setup required — this is a fully front-end application.
